# ttrpg_inventory_project
Inventory management project for ttrpgs:
- Text-based inventory management, with options to:
  - Store Items/Stats
  - Retrieve Items
  - Update information
  - View a list of inventory
- Utilizing JSON, list/dictionary comprehension
